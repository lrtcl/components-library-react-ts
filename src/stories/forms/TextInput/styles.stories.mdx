# Styles

The following CSS classes are available to target the component's elements:

| Class                              | Description                                          |
|------------------------------------|------------------------------------------------------|
| .mylib--textinput                  | Global container for the component                   |
| .mylib--textinput__label           | The input's label                                    |
| .mylib--textinput__required-text   | The label's visual indicator for a required input    |
| .mylib--textinput__input           | The input element                                    |
| .mylib--textinput__message         | Generic class for both helper text and error message |
| .mylib--textinput__helper          | Helper text                                          |
| .mylib--textinput__error-message   | Error message                                        |
| .mylib--textinput__counter         | The character counter                                |
