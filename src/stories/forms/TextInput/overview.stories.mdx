# Text input

<ul class="tagslist">
  <li class="tagslist__tag tagslist__tag_status_wip">WIP</li>
</ul>

Input element that creates a basic single-line text field.
Line-breaks are automatically removed from the input value.

ATTENTION: this is the correct form field to use for numerical values, when special formatting is needed (eg. credit card number) and/or when a spin button is not usable or recommended (eg. phone number).
